Questionário - Questões

Selecione os valores das características que ache necessário que seu SGBD possua.

Para um melhor resultado, marque apenas as alternativas que você assuma que sejam necessárias para seu SGBD.

Segurança

É preferivel que o SGBD realize criptografia dos dados armazenados?
É preferivel que o SGBD realize controle de acesso?
É preferivel que o SGBD seja capaz de manipular privilégios de usuários?
É preferivel que o SGBD consiga realizar backups?

Disponibilidade

É preferivel que o SGBD seja capaz de realizar Espelhamento?
É preferivel que o SGBD seja capaz de realizar Fragmentação?
O sistema necessita de alta tolerância a falhas?

Consistência e Integridade de Dados

É preferivel que o SGBD suporte transações no modelo ACID?
O sistema irá utilizar restrições ou limitações de regras de negócios?
É preferivel que o SGBD manipule os dados de maneira concorrente (travar ou a linha ou a tabela quando já estiver realizando uma operação)?

Usabilidade

O usuário faz questão de que a interface seja limpa e acessível?
Qual o nível de familiaridade do usuário com SGBDs?
É preferível que a interface do SGBD seja customizável?

Interoperabilidade

É preferivel que o SGBD seja “open source”?
O sistema irá se comunicar com terceiros (programas externos)?
Haverá uma situação em que o SGBD necessite ser executado em algum modo de compatibilidade?

Desempenho e escalabilidade

Seria problema o SGBD ser executado em uma máquina de hardware mais fraco?
O tempo de resposta é crítico?
O sistema irá trabalhar com uma massa de dados muito grande?
